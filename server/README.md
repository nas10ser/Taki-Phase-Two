# 🤖 بوت TAKI — دليل التشغيل (تيليجرام + واتساب)

البوت جاهز بالكامل (`bot.js`). يبقى فقط أن تضع **رمز البوت (Token)** الخاص بك ثم تشغّله.
كل الأسرار تُقرأ من متغيّرات البيئة — **لا يوجد أي رمز مكتوب داخل الكود**.

---

## ① احصل على رمز بوت تيليجرام (دقيقتان)

1. افتح تطبيق **تيليجرام** وابحث عن: **@BotFather** (الحساب الرسمي بعلامة ✔️).
2. أرسل له: `/newbot`
3. اكتب **اسم البوت** (مثلاً: تاكي TAKI)، ثم **معرّفاً** ينتهي بـ`bot` (مثلاً: `taki_deals_bot`).
4. سيعطيك **رمزاً (Token)** يشبه:
   `8123456789:AAH...الخ`
   👈 **هذا هو رمزك السرّي.** انسخه ولا تشاركه مع أحد علناً.

---

## ② ضع الرمز في ملف الإعدادات

1. في مجلد `server/` انسخ ملف القالب إلى ملف حقيقي:
   ```bash
   cp .env.example .env
   ```
2. افتح ملف `.env` والصق:
   - `TELEGRAM_BOT_TOKEN=` ← الرمز من BotFather
   - `TELEGRAM_WEBHOOK_SECRET=` ← أي نص عشوائي طويل تختاره أنت (للحماية)
   - `SUPABASE_URL=` و `SUPABASE_ANON_KEY=` ← من لوحة Supabase ▸ Project Settings ▸ API
   - `APP_URL=` ← عنوان موقعك (مضبوط مسبقاً على رابط الإنتاج)

   > ملف `.env` **لا يُرفع إلى GitHub** (مستثنى تلقائياً) — يبقى على جهازك/الخادم فقط.

---

## ③ التشغيل

### تجربة محلية سريعة (على جهازك)
```bash
cd server
npm install
npm start
```
سيعمل البوت على المنفذ 8080. (هذا للتجربة فقط — جهازك يجب أن يبقى مفتوحاً.)

### التشغيل الدائم (الطريقة الاحترافية — موصى بها)
البوت يحتاج خادماً يعمل 24/7. أسهل خيار مجاني/رخيص: **Railway** أو **Render**.

1. ادخل [railway.app](https://railway.app) أو [render.com](https://render.com) وسجّل بحساب GitHub.
2. اختر **New Project ▸ Deploy from GitHub repo** ثم اختر مستودع TAKI، ومجلّد الجذر `server`.
3. في إعدادات المشروع ▸ **Variables**، أضف نفس المتغيّرات الموجودة في `.env`.
4. سيعطيك الموقع رابطاً عاماً مثل: `https://taki-bot-xxxx.up.railway.app`

---

## ④ ربط الـWebhook (مرة واحدة بعد التشغيل الدائم)

استبدل `<TOKEN>` برمزك، و`<URL>` بعنوان الخادم من الخطوة ③، و`<SECRET>` برمز الحماية الذي اخترته:

```
https://api.telegram.org/bot<TOKEN>/setWebhook?url=<URL>/webhook/telegram&secret_token=<SECRET>
```
افتح هذا الرابط في المتصفّح مرة واحدة — إذا ظهر `"ok":true` فالبوت متّصل ويعمل ✅.

---

## ⑤ تأكّد أنه يعمل
- افتح بوتك في تيليجرام واضغط **Start** — يجب أن يردّ بقائمة تاكي.
- صفحة الفحص: افتح `<URL>/health` يجب أن تُظهر `ok`.

---

### 🔐 ملاحظات أمان
- لا تضع الرمز في أي رسالة عامة أو في الكود — فقط في `.env` أو متغيّرات المنصّة.
- لو تسرّب الرمز: ارجع لـ@BotFather ▸ `/revoke` ▸ واحصل على رمز جديد.
- البوت يستخدم مفتاح Supabase العام (anon) فقط — لا يملك صلاحيات تتجاوز سياسات الأمان (RLS).

> **جاهز؟** أعطني رمز تيليجرام (Token) ومتى ما رغبت أكمل لك الربط والنشر خطوة بخطوة.
